1) O que significa dizer que funções são entidades de primeira classe?
R: Significa que elas podem ser utilizadas como parametro de entrada,
parametro de saida, podem ser tipadas, criadas localmente

2) O que é uma função de alta ordem?
R: Uma função que recebe uma ou mais funções, e/ou
retorna uma ou mais funções

3) Quando podemos utilizar as funções map, filter e foldr?
R: Você utilizara uma das três quando a lista é processada por inteiro
Map: quando queremos aplicar uma função a cada elemento da lista,
de forma independente. Você "mapeia" cada elemento para outra função

Filter: Quando você seleciona os elementos de uma lista de acordo
com algo, por exmplo, selecionar os elementos pares de uma lista, 
você devera usar o filter

Fold_right: Quando queremos calcular algo de forma incremental,
analisando cada elemento de uma lista.

4) Qual estratégia foi discutida para projetar funções que 
recebem funções como parâmetro?
R: Primeiramente, você deve tentar identificar funções com corpo
parecido, semelhante.
Identificar qual a pequena mudança de uma para outra e criar parametros
Copiar o corpo, e, naquela pequena mudança entre as duas funções,
voce muda pelos parametros criados.
Depois você deverá escrever os exemplos, reutilizar os exemplos
das funções
Escrever o propósito
Escrever a assinatura
Substituir todo o corpo das funções iniciais pela chamada da nova
função

5) O que é uma variável livre?
R: Uma variável que não é global, não foi criada localmente
dentro da função, não é um parametro da função. Ou seja,
uma variável criada dentro de outra função, que pode ser utilizada
na função atual
